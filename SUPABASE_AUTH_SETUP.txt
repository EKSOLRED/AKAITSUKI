1) Открой js/config/supabase.config.js
2) Поставь:
   enabled: true
   url: 'https://YOUR_PROJECT.supabase.co'
   anonKey: 'YOUR_SUPABASE_ANON_KEY'
   redirectTo: 'https://YOUR_DOMAIN/reset-password'
3) В Supabase создай таблицу profiles:
   id uuid primary key references auth.users(id) on delete cascade
   email text
   name text
   role text default 'user'
   created_at timestamptz default now()
4) Включи Email auth в Supabase Auth.
5) Для восстановления пароля добавь redirect URL в Auth > URL Configuration.
