## Что ещё остаётся перед полноценным live-подключением Supabase

Главное ограничение текущей архитектуры: она синхронная, а Supabase — асинхронный.

Для полноценного переключения на Supabase без костылей нужен следующий шаг:
- перевести репозитории/API-слой на `async/await`
- загрузку данных страницы сделать асинхронной
- вынести auth в Supabase Auth
- убрать хранение локальных паролей из production-режима

То есть архив **подготовлен к подключению**, но не включает «боевое включение Supabase в один флаг» — это уже отдельный этап интеграции.
